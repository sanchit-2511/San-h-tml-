# San(h!tml)

- SIES(NERUL) Data Science PortalA clean, informative web portal designed for the Department of FY.BSc Data Science at SIES (Nerul) College of Arts, Science, and Commerce.
- This project was developed as a practical application of Web Technology fundamentals, focusing on structured HTML5 and responsive-style CSS.

## 🚀 Project Overview

- This website serves as an informational landing page for prospective and current Data Science students.
- It details the curriculum, career opportunities, and college contact information in a structured, easy-to-navigate format.

## ✨ Features

- Interactive Curriculum Section: Uses HTML5 <details> and <summary> tags for a clean, "accordion-style" view of subjects.
- Data Presentation: Includes a comprehensive table outlining qualifications, recruiting companies, and job positions.
- Responsive Layout: Utilizes CSS Flexbox for a side-by-side content layout (Subjects vs. Info Table).
- Themed Design: A dark-mode aesthetic using a custom green-and-teal color palette (#32746d, #9ec5ab).
- User Feedback Form: A stylized "About You" footer form for student inquiries.
- Navigation: A fully functional navigation bar with a search interface and FontAwesome integration.

## 🛠️ Technologies Used

- HTML5: Semantic structure, tables, and interactive elements.
- CSS3: Flexbox, custom positioning, hover transitions, and Google Font styling.
- FontAwesome: For social and utility icons.
